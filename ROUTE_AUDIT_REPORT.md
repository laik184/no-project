## Bootstrap Sequence (Before Fix)

1. `bootstrapMemory()` — memory platform
2. `heartbeatManager.start()` — **module-level side effect** in chat/index.ts import
3. `app.use('/api/chat', chatRouter)` — chat routes
4. `app.use('/', buildSseRouter())` — SSE at root ⚠️
5. `app.use('/api', consolePipeline)` — console
6. `app.use('/api', previewPipeline)` — preview
7. `app.use('/api/orchestration', createOrchestrationRouter())` — orchestration
8. `app.use('/api', projectsRouter)` — projects
9. `app.use('/api/run', runStartRouter)` — run aliases
10. `chatOrchestrator.attachWebSocket(server)` — WS
11. `initOrchestration()` — orchestrator init
12. `chatOrchestrator.startPersistence()` → `heartbeatManager.start()` ⚠️ second call
13. `server.listen()`
