---
name: Settings row system
description: The Settings Modal uses a user-defined 69px standard for its reusable rows and cards.
---

The Settings Modal standard row height is 69px on desktop. Shared form fields, toggles, profile summaries, provider rows, action rows, informational rows, temperature, and appearance controls should preserve this rhythm when future UI adjustments are made.

**Why:** The product direction prioritizes visual consistency across all settings categories without redesigning the information architecture or sacrificing readability and accessibility.

**How to apply:** Reuse the existing settings row/form-row classes and keep content readable on smaller screens by allowing natural height expansion where a fixed 69px row cannot safely contain the content.
