IMPORTANT ARCHITECTURE RULES
==================================================

Tool Dispatcher

Coding Tools

==================================================
TOOL RESPONSIBILITY RULES
==================================================

Each tool MUST do ONLY ONE thing.

Examples:

generate-react-page.ts
→ ONLY React page generation

generate-controller.ts
→ ONLY backend controller generation

generate-jwt-auth.ts
→ ONLY JWT auth generation

NO mixed generation logic.

- metadata
- handler
- validation schema

3. Use:
- ToolExecutionContext
- ToolExecutionResult

4. Return normalized outputs
