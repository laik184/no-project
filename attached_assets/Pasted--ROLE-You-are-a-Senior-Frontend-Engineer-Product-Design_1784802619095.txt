# ROLE

You are a Senior Frontend Engineer, Product Designer, UX Engineer, and React Architect.

Your task is to improve ONLY the Home Page (/).

Do NOT modify any other page.

Do NOT modify backend APIs.

Do NOT modify unrelated components.

Do NOT create fake data.

--------------------------------------------------

GOAL

Fix every Home Page issue identified in the frontend audit report while preserving the existing design language.

The final page must feel production-ready.

--------------------------------------------------

Read every imported component.

Read every hook.

Read every state.

Read every event handler.

Understand the page completely before making changes.

Never skip any file.
