You are a senior principal autonomous systems engineer and TypeScript platform architect.

Your task is to implement a production-grade "server/agents/verifier/" agent for an autonomous AI coding system.

IMPORTANT:
This is NOT a tool layer.

This is ONLY an AGENT orchestration layer.

The verifier agent must:

- coordinate verification flows
- orchestrate validation execution
- manage verification sessions
- control retries/recovery
- route verification workflows
- analyze failures
- manage verification state
- coordinate tools through dispatcher ONLY

The verifier agent MUST NOT:

- directly run builds
- directly execute terminal commands
- directly execute Playwright/browser logic
- directly execute TypeScript compiler
- directly perform filesystem operations
- directly validate runtime internally

ALL actual execution MUST happen through:

- dispatcher-client.ts
- tool-coordinator.ts
- centralized tool dispatcher
