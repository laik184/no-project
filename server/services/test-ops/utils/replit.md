# Utils Folder

## Purpose
Contains L3 helpers used by L2 agents.

## Files
- `command-builder.util.ts`: builds framework-specific command/args.
- `output-parser.util.ts`: parsing helpers for counts, coverage, failures.
- `logger.util.ts`: log message helpers.
- `timeout.util.ts`: timeout wrapper utility.

## Import Rule
Utilities do not import agents or orchestrator.
